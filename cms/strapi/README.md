# Strapi CMS plan for Mahadev Adi Guru

Use Strapi when you want a full admin panel with PostgreSQL, media library, roles, draft/publish and APIs.

Recommended content types:
- Article
- Mantra
- Temple
- Jyotirlinga
- Festival
- YouTube Video
- Quote

Deployment options:
- VPS / Docker Compose for Strapi + PostgreSQL
- Frontend stays on Cloudflare Pages
- Images can later move to Cloudflare R2 or S3-compatible storage
