# Mahadev Adi Guru Platform

This is a production-style starter package for mahadevadiguru.com.

## What is included
- `site/` — Cloudflare Pages-ready website
- `site/admin/` — Decap CMS starter config
- `cms/strapi/` — Strapi content-type schema examples
- `cms/directus/` — Directus collection plan
- `docs/DEPLOYMENT.md` — Cloudflare deployment guide

## Current deployment
For immediate hosting, upload the `site` folder contents to Cloudflare Pages using Direct Upload.

## Recommended roadmap
1. Launch static site on Cloudflare Pages.
2. Connect GitHub and Decap CMS for browser-based editing.
3. Add Strapi + PostgreSQL when you need full admin roles, media workflows, draft/publish and API-driven content.
